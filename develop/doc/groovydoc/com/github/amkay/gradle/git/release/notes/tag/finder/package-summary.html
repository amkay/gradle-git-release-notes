Package com.github.amkay.gradle.git.release.notes.tag.finder

Provides the strategies for finding the Git tag of the latest release version of the project to which the plugin is applied.

The entry point to this package is the interface TagFinder.

  • Interface Summary 
    Interface Description
    TagFinder The interface for all strategies used to find the Git tag of the latest release version.
  • Class Summary 
    Class Description
    GitflowVersionTagFinder This strategy for finding the Git tag of the latest release version is used if the version of the project the plugin was applied on has a normalVersion property.
    NearestTagFinder This strategy for finding the Git tag of the latest release version is used if no other strategies find a matching Git tag.
    VersionTagFinder This strategy for finding the Git tag of the latest release version is used if the version of the project the plugin was applied on is a normal String.
